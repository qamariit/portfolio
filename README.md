### Run Locally

To run **qamariit-portfolio** locally, run this command on your git bash:

Linux and macOS:

```bash
sudo git clone https://github.com/qamariit/portfolio.git
```

Windows:

```bash
git clone https://github.com/qamariit/portfolio.git
```
